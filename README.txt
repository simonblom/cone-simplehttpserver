Pyttewebb 1.0 av Simon Blom
    version: 2016-02-19
---------------------------
Programmet skapar en simpel 
HTTP server.
---------------------------
Placera .jar filen tillsam-
mans med resurser till
servern.
Kör kommandotolken och
anropa: 
"cd /YOURFOLDER/" där
YOURFOLDER är mappen med
.jar filen. Anropa därefter
"java -jar pyttewebb.jar"
för att starta servern.
---------------------------
